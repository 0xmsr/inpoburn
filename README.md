# 🔥 Inpoburn Hub | ERC20 Burner & IB Token Claimer

**Inpoburn Hub** is a decentralized application (dApp) built on the Base Network that allows users to burn any ERC20 tokens in exchange for **IB Tokens**. This project is designed with a sleek, high-performance "fire-themed" UI and a robust integration with the Blockscout API for real-time token detection.

## 🚀 Features

- **Multi-Token Detection**: Automatically scans user wallets for ERC20 tokens using Blockscout V2 API (with auto-pagination support).
- **One-Click Burn**: Streamlined process to Approve and Burn any unwanted tokens.
- **IB Token Rewards**: Receive 1 IB Token for every burn transaction.
- **Real-time Burn Counter**: Live tracking of the total IB supply directly from the smart contract.
- **Responsive "Terminal" UI**: Unique aesthetic with flame animations and "Share Tech Mono" typography.
- **Multilingual Support**: Toggle between English and Bahasa Indonesia.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Custom animations), Vanilla JavaScript.
- **Web3**: [Ethers.js v6](https://docs.ethers.org/v6/).
- **API**: Blockscout V2 (Base Explorer).
- **Network**: Base Mainnet (Chain ID: 8453).
